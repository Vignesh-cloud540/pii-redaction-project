# PII Redaction & Safe Output Validator

## 📌 Description
This project detects and masks sensitive data like emails, phone numbers, and Aadhaar numbers using FastAPI.

## 🚀 Features
- Detects PII data
- Masks sensitive information
- Secure output validation

## 🛠 Technologies Used
- Python
- FastAPI
- Regex

## ▶️ How to Run
uvicorn app.main:app --reload
